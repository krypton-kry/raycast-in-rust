# Raycast
 A Raycast example written in rust with macroquad
### Showcase
![RayCast in Rust](https://imgur.com/download/XO8CHnK/)
 ### How to run
    git clone https://github.com/krypton-kry/raycast-in-rust.git
    cd raycast-in-rust
    cargo run
